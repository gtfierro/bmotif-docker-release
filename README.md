These steps are tested on Debian-based Linux (Ubuntu, Debian) and macOS systems.

1. Install [docker compose](https://docs.docker.com/compose/install/)
2. Clone this repository: `git clone https://github.com/gtfierro/bmotif-docker-release`
3. From inside the cloned repository, run: `docker compose up`
4. Navigate to `http://localhost:4200` in your web browser to access the application.
5. Follow the directions in `BuildingMOTIF-Demo-Slides.pdf` to use the application. Use the files in `demo-files` folder to test the application.
